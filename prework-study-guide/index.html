HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section elements are semantic elements that define a section in a document that contains thematic content
  • The h1 is the largest text and h6 the smallest
  • the p element represents a paragraph
  • the ul, ol and li elements represent unordered lists, ordered lists and list items.
  • The br element creates a line of empty space pr line break.
  • The img element contains information about images that are displayed on the web.
  • The scr attribute defines the location of the image.
