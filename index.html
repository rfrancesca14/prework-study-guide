HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section elements are semantic elements that define a section in a document that contains thematic content
  • The h1 is the largest text and h6 the smallest
  • the p element represents a paragraph
  • the ul, ol and li elements represent unordered lists, ordered lists and list items.
  • The br element creates a line of empty space pr line break.
  • The img element contains information about images that are displayed on the web.
  • The scr attribute defines the location of the image.

CSS

  • CSS defines the colors, aesthetics, visual layout, and organization. We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • The border property assigns a border
  • The box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

GiT

  • Git allows users to track code changes and manage their project using simple commands.
  • A repository is used to contain a project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch -a checks for existing branches

JavaScript

  • A variable is a named container that allows us to store data in our code./li>
  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, to declare a variable, we use the var keyword. After the var keyword, we give the variable a name. This name must be unique because it is how we will reference the information stored in the variable in our code.
  • we use an assignment operator = to assign the specific data that we want stored in our variable.
  • We can use the console.log method to output a message to the web console by adding an argument.
